# Age Calculator
This project is a simple age calculator built using HTML, CSS, and JavaScript. It takes a user's birthdate as input and calculates their age in years.

## Project Structure
index.html - The main HTML file containing the structure of the page.

styles.css - The CSS file for styling the elements.

script.js - The JavaScript file containing the logic for calculating the age.

### Features
Input field for birthdate.

Button to calculate age.

Displays the calculated age.
